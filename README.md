# AI Mock Interview Coach

A React + Vite + Tailwind CSS web application that offers students free, AI-powered mock interviews. Built with Gemini 1.5 Flash.

## Setup

1. Copy `.env.example` to `.env.local`:
   ```bash
   cp .env.example .env.local
   ```
2. Get a free Gemini API Key from [Google AI Studio](https://aistudio.google.com/app/apikey) and insert it in `.env.local`.

3. Install dependencies:
   ```bash
   npm install
   ```

4. Run locally:
   ```bash
   npm run dev
   ```

## Stack
- React 18
- Vite
- Tailwind CSS
- Native Web Speech API
- Gemini API (Free tier)
